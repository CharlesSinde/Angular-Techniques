What I learned during NgConf 2017! -HUGE WIP


Each tab explains a different concept or idea learned at Ngconf. The words will describe what is happening then you can dive into

the code to check out what is happening behind the scenes.


If you are veiwing this from a remote site then first step:

Go to https://github.com/CharlesSinde/NgConf2017 to clone the repo. Next run a "npm install npm" in the project folder.

Ng serve in cmd while under the NgConf2017 folder. *Note: make sure to have node 6.10 or higher and CLI 1.0.0 or higher
